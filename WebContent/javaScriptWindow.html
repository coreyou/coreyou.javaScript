alert: 彈出訊息框

confirm: 彈出詢問框

按下確定就傳回 true，按下取消就傳回 false。

prompt(message, inputDefault): 彈出輸入框

message是輸入框的訊息，inputDefault是預設的輸入資料，最後傳回輸入了的資料

window.open(url, name, features): 開啟新視窗

url是要載入的URL，如果是空字串，會開啟空視窗；name是新視窗名稱，為<a>、<form>的target屬性；features是選擇性屬性。

features的屬性 說明
height 視窗高度pixel
width 視窗寬度pixel
location 位置欄(值可以為: 1或0、yes或no)
directories 分類目錄列(值可以為: 1或0、yes或no)
menubar 主選單(值可以為: 1或0、yes或no)
scrollbars 捲軸(值可以為: 1或0、yes或no)
status 狀態列(值可以為: 1或0、yes或no)
toolbar 工具列(值可以為: 1或0、yes或no)
resizable 可否改變視窗大小(值可以為: 1或0、yes或no)

window.close(): 關閉視窗

setTimeout(statement, delay): 在某個時段過後自動執行某敘述

statement是字串，為要執行的JavaScript敘述；delay是微秒。
